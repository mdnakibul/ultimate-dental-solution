# Ultimate Dental Solution 
by

It is a dental hospital management system.You can book an appointment, see your bookings, doctors can see their dashboard and many more

## Features : 
1. Authentication
2. Patient Dashboard
3. Doctor Dashboard
4. Role management
5. Dynamic routing
